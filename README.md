# sydney-transport-dashboard

basic departure board for sydney train stations.

## Getting Started
1. Clone the repository
`git clone https://github.com/kittymaru/sydney-transport-dashboard.git`
`cd sydney-transport-dashboard`

2. Install dependencies with `npm install` in /backend and /web-frontend.

3. Create a .env file in /backend with Transport NSW API key. It must be in the format
`YOUR_API_KEY=key`

4. Start backend server with `node server.js`.

5. Start frontend server with `npm run dev`.

6. Open `http://localhost:5173` in browser.
